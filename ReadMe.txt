HybridNTierECommerce


HybridNTierECommerce is adapted from the N-Tier architecture utilizing the interface structure that minimizes dependencies in Onion architecture. The primary goal of the project is to establish a well-structured architecture and perform CRUD operations effectively. A pre-made template was used for the UI, and necessary enhancements have been made. The main objectives of the project have been achieved, and additional improvements will be implemented as time allows.
